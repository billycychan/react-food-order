## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/billycychan/react-food-order.git
   ```

2. Navigate to the project directory:
   ```sh![alt text](image.png)
   cd react-food-order
   ```

3. Run the backend server:
   ```sh
   cd backend
   npm start
   ```

4. Run the project from the root:
   ```sh
   npm run dev
   ```
